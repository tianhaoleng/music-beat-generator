# 🎵 Music Beat Generator - 音乐伴奏生成器

一个简单快速的音乐伴奏生成工具，适合视频创作者、音乐爱好者使用。

[English](#english-version) | 中文

## ✨ 功能特性

- 🎵 **5种音乐风格**：说唱、电子、Trap、深度电子、Lofi
- 🎚️ **自定义参数**：BPM（60-180）、长度（30秒-5分钟）、强度（1-10）
- 🎧 **实时预览**：生成后立即听效果
- 📥 **多格式导出**：MP3、WAV、MIDI
- ⚡ **快速生成**：秒级响应
- 🎯 **傻瓜式操作**：0基础可用
